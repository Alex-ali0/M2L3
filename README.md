# Телеграм-бот Викторина с котиками

Этот бот — забавная викторина о котиках, которая отправляет вопросы с вариантами ответов и подсчитывает количество правильных ответов пользователя.

## Особенности
- Отправка изображения вместе с вопросом
- Использование inline-клавиатуры для выбора ответа
- Подсчет очков и вывод финального результата
- Простая структура кода на Python с использованием библиотеки `pyTelegramBotAPI` (telebot)

## Структура проекта

- `bot.py` — основной файл бота, управляет логикой взаимодействия
- `logic.py` — содержит класс `Question` и список вопросов (`quiz_questions`)
- `config.py` — хранит ваш токен бота (необходимо создать файл с переменной `token`)
- Изображение `3524032.png` — используется для отображения перед вопросом
